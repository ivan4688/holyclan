# 🛡️ HolyClan (Fabric 1.20.1)
## HolyClan — это приватный клиентский мод для сервера HolyWorld, созданный для быстрого и автоматического поиска кланов с запрещенными названиями (маты, политика, оскорбления, обход фильтров).
## ✨ Функции мода
- Мгновенный сканер: Читает память интерфейса. Страницы без нарушителей пролистываются за секунду!
- Авто-скриншоты: Мод сам наводит мышку на голову нарушителя, ждет прогрузки создателя клана, делает скриншот и заливает его на FreeImage.host.
- Готовый отчет: На выходе вы получаете файл со списком формата: Создатель | Название | Ссылка на скриншот.
- DEV-режим: Возможность сдампить (сохранить) вообще все названия кланов со страниц для улучшения фильтра.
## ⚙️ Установка и Настройка
- Установите Fabric Loader для версии 1.20.1.
- Поместите мод в папку .minecraft/mods.
- Запустите игру. При первом запуске в папке .minecraft автоматически создадутся нужные файлы.
## 🔑 ШАГ 1: Получение API ключа (ОБЯЗАТЕЛЬНО)
#### Чтобы мод мог автоматически загружать скриншоты в интернет, нужен бесплатный ключ от фотохостинга:
#### Перейдите на сайт: https://freeimage.host/page/api
#### Зарегистрируйтесь (или войдите).
#### На странице API скопируйте свой длинный API Key.
#### Зайдите в Майнкрафт на сервер и пропишите команду:
#### /holyclan setapi ВАШ_КЛЮЧ
## 🚀 Как пользоваться?
Есть два способа запустить сканирование:
- Способ 1 (Командой):
Просто пропишите /holyclan find. Мод сам откроет меню кланов и начнет сканировать все страницы с первой до последней.
- Способ 2 (Горячей клавишей):
Откройте /clan list вручную на нужной странице и нажмите клавишу O (английскую). Скан начнется прямо с текущей страницы.
## ⏳ Во время сканирования:
- Не трогайте мышку и не закрывайте инвентарь!
- Мод будет сам листать страницы. Если он найдет нарушителя, он остановится на секунду, чтобы сделать скриншот, и продолжит дальше.
- По завершении (когда страницы закончатся) в чате появится уведомление.
## 📁 Где искать результаты?
- Зайдите в папку .minecraft. Там появится файл holyclan_scanned.txt. Откройте его, и вы увидите готовый список нарушителей со ссылками на доказательства.
## ⌨️ Команды и Управление
### Основные команды
- /holyclan find — Запустить обычный поиск нарушителей.
- /holyclan dev — Запустить режим разработчика (сохраняет ВСЕ названия кланов со страниц в файл holyclan_dev.txt, удобно для пополнения словаря матов).
- /holyclan reload — Перезагрузить текстовые файлы черного списка и исключений прямо во время игры + очистить файлы отчетов.
- /holyclan setapi <ключ> — Установить/поменять API-ключ для скриншотов.
## 📂 Файлы конфигурации (лежат в .minecraft)
### Вы можете настраивать мод, просто редактируя текстовые файлы блокнотом. Чтобы изменения вступили в силу без перезапуска игры, пропишите /holyclan reload.
- holyclan_blacklist.txt — Черный список. Добавляйте сюда корни матов, плохие слова, запрещенные названия софтов и т.д.
- holyclan_exclusions.txt — Белый список (исключения). Слова отсюда будут вырезаться из названия клана перед проверкой. Спасает от ложных банов (например, слово "оскорблять" содержит корень "ебля", добавьте "оскорбл" в исключения).
- holyclan_scanned.txt — Сюда падают результаты обычного сканирования с доказательствами.
- holyclan_dev.txt — Сюда падают дампы страниц при использовании DEV-скана.
- holyclan_api.txt — Здесь хранится ваш API-ключ.
## ⚠️ Частые проблемы
- "Ошибка! Возможно, неверный API ключ" — Вы неправильно ввели ключ от FreeImage. Скопируйте его заново и пропишите /holyclan setapi.
- Мод пропускает маты? — Откройте holyclan_blacklist.txt и добавьте туда корень пропущенного слова, затем напишите /holyclan reload.
- Мод "зависает" на клане? — Если интернет медленный, отправка картинки может занять пару секунд. Просто подождите, мод сам продолжит работу, как только фото загрузится.
